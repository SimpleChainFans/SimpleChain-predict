# SimPre
[中文](README.md)
## 基于simplechain区块链智能合约技术和Vue框架开发的投票预测平台
平台完整体验采用安装[SimPlug]浏览器插件的解决办法。
## 功能与优化
- [x] 连接区块链
- [x] 创建用户
- [x] 个人信息展示
- [x] 投票事件展示
- [x] 创建事件
- [x] 事件投票
- [x] 事件汇报
- [x] 申请奖励
- [x] 不同颜色展示事件状态
- [x] 展示用户所以操作记录
- [x] Other

## 所用技术
- VueCli 3.x
- Vue 2.x
- Vuex
- Vue Router
- [Web3.js](https://github.com/ethereum/web3.js/)
- ES6
- Node.js
- axios
- webpack
- 其他

注意：在VueCli 2.x中公共文件放入static文件夹，而在VueCli 3.x版本中请放入public文件夹，否则axios无法找到文件。该项目采用VueCli 3.x。
